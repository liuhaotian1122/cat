## 🌟 项目特色功能

### 1. **创意十足的猫咪造型**
项目通过复杂的 CSS 几何图形组合，创建了一只可爱的倒立猫咪。猫咪包含以下部分：  
- 头部（Head）- 旋转45°的头颅造型  
- 身体（Body）- 椭圆形的身体结构  
- 尾巴（Tail）- 灵活摇摆的长尾巴  
- 四肢（Paws & Legs）- 动画化的爪子和腿部  
- 面部细节（Face Details）- 眼睛、鼻子、胡须、眉毛、耳朵等

### 2. **丰富的动画效果**
项目定义了12+种关键帧动画，实现流畅的动画表现：

| 动画名称 | 目标元素 | 效果描述 |
|---------|--------|--------|
| `shake` | 主体 | 上下抖动动画（6px振幅） |
| `shadow` | 阴影 | 阴影宽度变化和位置调整 |
| `tail` | 尾巴 | 尾巴旋转（0°-38°） |
| `track` | 脚印 | 脚印轨迹横向移动 |
| `eye` | 眼睛 | 眼睛上下眨动 |
| `mustache_l` | 左胡须 | 胡须向左旋转（-10°） |
| `mustache_r` | 右胡须 | 胡须向右旋转（-10°） |
| `ear_l` | 左耳 | 左耳旋转（-30°） |
| `ear_r` | 右耳 | 右耳旋转（30°） |
| `paw_l` | 左爪 | 左爪摆动动画 |
| `paw_r` | 右爪 | 右爪摆动动画 |

### 3. **页面自动跳转**
HTML 中包含 JavaScript 代码，页面加载后 2 秒自动跳转到作者个人站点：
```javascript
setTimeout(function() {
  window.location.href = 'https://liuhaotian1122.github.io/my-pages/';
}, 2000);
```

### 4. **个人品牌展示**
页面底部展示作者信息：
- 头像显示  
- "个人站" 标题  
- "Powered by Haotian" 版权信息

## 🎬 动画流程分析

### 动画参数
- **动画时长**: 0.45 秒（大多数动画）  
- **动画循环**: infinite（无限循环）  
- **动画时序**: linear（线性）  
- **总周期**: 450 毫秒

### 动画时间序列

```
0ms    ├─ 爪子1动画开始
       ├─ 尾巴动画开始
       ├─ 胡须动画开始
       └─ 眼睛动画开始

112ms  ├─ 爪子1运动到最低点
       ├─ 尾巴旋转到最大角度
       └─ 眼睛眨动最明显

225ms  ├─ 脚印轨迹延迟启动（延迟225ms）
       └─ 动画反向开始

337ms  ├─ 爪子2运动到最低点
       └─ 另一套视觉元素到达中点

450ms  └─ 所有动画循环回到初始状态
```

## 💡 CSS 高级技巧

### 1. 使用 currentcolor 变量
```css
color: #000;  /* 在容器设置 */
background: currentcolor;  /* 继承父级颜色 */
```

### 2. 复杂的 border-radius
```css
border-radius: 55px/100px 100px 65px 65px;  /* 水平/竖直 */
```

### 3. 伪元素创建细节
```css
.element:before { /* 创建前置元素 */ }
.element:after { /* 创建后置元素 */ }
```

### 4. box-shadow 堆叠
```css
box-shadow: 50px 0 #616161, 100px 0 #616161, ...;  /* 多层阴影 */
```

### 5. transform 叠加
```css
transform: rotateY(180deg) rotateZ(-10deg);  /* 多轴旋转 */
```
